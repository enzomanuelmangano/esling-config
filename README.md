# eslint-config

[![Version][version-badge]][package]

## Installation

```
yarn add --dev eslint @reactiive/eslint-config
```

## Usage

Add to your eslint config (`.eslintrc`, or `eslintConfig` field in `package.json`):

```json
{
    "extends": "@reactiive/eslint-config"
}
```
